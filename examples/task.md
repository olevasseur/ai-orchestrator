# Task: Add a simple calculator module

Add a `calculator.py` module to this repository with the following functions:
- `add(a, b)` — returns a + b
- `subtract(a, b)` — returns a - b
- `multiply(a, b)` — returns a * b
- `divide(a, b)` — returns a / b, raises ValueError if b is 0

Also add `tests/test_calculator.py` with pytest tests for each function,
including an edge-case test for division by zero.

Keep the implementation minimal and idiomatic Python.
